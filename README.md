# Calculadora de IMC

Este projeto é uma aplicação simples em Dart para calcular o Índice de Massa Corporal (IMC).

## Funcionalidades

- Calcula o IMC com base na altura e peso fornecidos pelo usuário.
- Exibe a categoria do IMC (Abaixo do peso, Peso normal, Sobrepeso, Obesidade).

## Como usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/calculadora-imc-dart.git
   ```

2. Navegue até o diretório do projeto:
   ```bash
   cd calculadora-imc-dart
   ```

3. Execute o código Dart:
   ```bash
   dart run
   ```
