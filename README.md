# Calculadora de IMC

Este projeto é uma aplicação simples em Dart para calcular o Índice de Massa Corporal (IMC).

# Funcionalidades

- Calcula o IMC com base na altura e peso fornecidos pelo usuário.
- Exibe a categoria do IMC (Abaixo do peso, Peso normal, Sobrepeso, Obesidade).

# Calculadora de IMC (Flutter / Dart)

Este projeto é uma aplicação desenvolvida em **Flutter** utilizando **Dart**, criada para calcular o **Índice de Massa Corporal (IMC)** de forma simples e rápida.  
O usuário informa seu peso e altura, e o app retorna a classificação do IMC.

---

# Tecnologias utilizadas

- **Flutter**
- **Dart**
- Widgets: `StatefulWidget`, `TextFormField`, `ElevatedButton`, `AppBar`, entre outros.
- Validação de formulários com `Form` e `GlobalKey`.

---

# Funcionalidades

-  Calcula o IMC com base no peso e altura informados.
- Exibe a categoria do IMC:
  - Abaixo do peso  
  - Peso normal  
  - Acima do peso  
  - Obesidade  
-  Botão para limpar os dados e reiniciar o formulário.
-  Interface simples e responsiva.
